## 📕️ Como baixar o projeto

```bash
    #Clonar o repositório
    $ git clone  https://github.com/Liliamsn

    #Entrar no diretório
    $ cd to-do-list

    #Instalar o live-server globalmente
    $ npm install -g live-server

    #Inicializar o projeto
    $ live-server
```
